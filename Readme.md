Matmath
=====================

A library for math functions.

Design goals
-----------------
* Simplicity to use
* High performance
* Similarity to mathmatical notations of similar operations (when applicable)


How to use
-----------------
Add matmath into your project. Then add matmath/include to the include directory of your program.
